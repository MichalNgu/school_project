# **Název projektu**

**Auta značky Audi**

# **Žák**
Michal Nguyen

# **Stručný popis tématu (3–5 vět)**

Webová prezentace zaměřená na vybrané modely automobilů značky Audi. Stránka bude mít formu one-page webu s několika sekcemi, které návštěvníkům představí historii značky, informace o modelech, technologické inovace a aktuální nabídku vozů. Obsah bude z velké části načítán dynamicky z JSON souborů pomocí AJAXu. Součástí bude kontaktní formulář pro zájemce o testovací jízdu. Cílem projektu je vytvořit moderní, přehledný a responzivní web s důrazem na práci s daty.

# **Cílová skupina**

Zájemci o koupi vozu Audi, fanoušci automobilů, lidé hledající informace o technologiích Audi a návštěvníci zvažující testovací jízdu nebo návštěvu autosalonu.

# **Návrh obsahových sekcí**

1. **O značce Audi** – stručná historie automobilky, filozofie a zaměření.
2. **Modelová řada Audi** – charakteristika vybraných modelů (A3, A4, A6, Q7, e-tron…), dynamicky načítaná z JSON.
3. **Technologie Audi** – popis systémů jako quattro, Audi AI, MMI, bezpečnostní prvky.
4. **Aktuální nabídka vozů** – dynamicky načítané modely s fotografiemi, cenami a parametry.
5. **Kontakt / testovací jízda** – kontaktní údaje a jednoduchý formulář.

# **Návrh designu / wireframe**

- Barevné ladění v prémiových barvách (černá, tmavě šedá, červená).
- Moderní technický vzhled inspirovaný značkou Audi.
- Karty modelů v responzivním gridu (2–4 sloupce).
- UI prvky: stíny, jemné přechody, zaoblené prvky, Tailwind komponenty.

# **Použité technologie**

- **Frontend:** HTML, CSS / Tailwind CSS, JavaScript  
- **Data:** JSON soubory pro obsah sekcí + JSON seznam modelů  
- **AJAX:** načítání sekcí a nabídky modelů  
- **Backend:** jednoduchý PHP endpoint pro zpracování formuláře  
- **Verzování:** Git + GitHub
